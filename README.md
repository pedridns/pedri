# Pedri-dnstt
DNS SERVER Scanner For DNSTT Tunneling

Install pedridns using wget:

$ wget https://raw.githubusercontent.com/pedridns/pedri-dnstt/main/pedridns    

$ chmod +x pedridns     

Usage:                     

$ ./pedridns -h
